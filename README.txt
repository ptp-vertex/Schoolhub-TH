SchoolHub - โครงสร้างไฟล์หลังแยกส่วน
=======================================

ไฟล์เดิม SchoolHub_fixed__3_.html (36,000 บรรทัด / ~1.9MB ไฟล์เดียว)
ถูกแยกออกเป็น:

  index.html          -> โครงหน้าเว็บหลัก (HTML markup ทั้งหมด)
  css/xxx.css         -> ไฟล์ CSS แต่ละก้อน (98 ไฟล์ ตามลำดับเดิมในไฟล์ต้นฉบับ)
  js/xxx.js           -> ไฟล์ JavaScript แต่ละก้อน/แต่ละ patch (123 ไฟล์ ตามลำดับเดิม)

วิธีการแยก (สำคัญ เพื่อยืนยันว่าใช้งานได้เหมือนเดิม 100%)
---------------------------------------------------------
- ทุก <style>...</style> block ในไฟล์ต้นฉบับ ถูกดึงเนื้อหาออกมาเป็นไฟล์ .css
  แยกไฟล์ แล้วแทนที่ตำแหน่งเดิมด้วย <link rel="stylesheet" href="css/....css">
  โดยเรียงลำดับไฟล์เหมือนเดิมทุกประการ (ลำดับ CSS มีผลต่อ specificity จึงต้อง
  คงลำดับเดิมไว้ ซึ่งสคริปต์ที่ใช้แยกไฟล์ได้ตรวจสอบแล้วว่าลำดับตรงกัน)

- ทุก <script>...</script> ที่เป็นโค้ด inline (ไม่ใช่ CDN เช่น TailwindCSS,
  Font Awesome, xlsx, qrcode, tesseract) ถูกดึงเนื้อหาออกมาเป็นไฟล์ .js
  แยกไฟล์ แล้วแทนที่ตำแหน่งเดิมด้วย <script src="js/....js"></script>
  (คง attribute id และ type เดิมไว้ครบ เช่น type="module" สำหรับสคริปต์ที่ใช้
  Firebase) โดยเรียงลำดับและตำแหน่งเดิมทุกประการ เพราะโค้ดหลายจุดเป็น patch
  ที่ต้องรันตามลำดับก่อน-หลัง (บาง patch เขียนทับฟังก์ชันที่ script ก่อนหน้า
  ประกาศไว้)

- สคริปต์จาก CDN (TailwindCSS, cdnjs Font Awesome, xlsx-js-style, qrcodejs,
  tesseract.js) ไม่ถูกแตะต้อง ยังคงเป็น <script src="https://...">
  เหมือนเดิมทุกประการ

- ได้ตรวจสอบแบบอัตโนมัติแล้วว่า:
  1) โครงสร้าง HTML ภายนอกส่วน <script>/<style> เหมือนต้นฉบับ 100%
     (byte-to-byte เท่ากันทุกตัวอักษร)
  2) เนื้อหาไฟล์ .js และ .css ทั้ง 221 ไฟล์ ตรงกับเนื้อหาต้นฉบับทุกตัวอักษร
     ไม่มีการตัดขาดหรือคลาดเคลื่อน

หมายเหตุ: มีไฟล์ js/007.js ขนาดใหญ่ (~440KB) ซึ่งเป็นโค้ดแอปหลัก (ใช้ Firebase,
type="module") ไฟล์นี้จำเป็นต้องอยู่รวมกันเป็นก้อนเดียว เพราะเป็น ES Module
ที่มีการอ้างอิงตัวแปร/ฟังก์ชันเชื่อมโยงกันภายในไฟล์เดียวกัน การแยกไฟล์นี้ต่อ
อีกโดยไม่วิเคราะห์โค้ดอย่างละเอียดมีความเสี่ยงที่จะทำให้แอปพลิเคชันพัง จึงเก็บ
ไว้เป็นไฟล์เดียวเพื่อความปลอดภัย ส่วนไฟล์อื่น ๆ ที่เป็น patch ย่อย ๆ (ทั้งหมด
~120 ไฟล์ ส่วนใหญ่มี id กำกับชัดเจนบอกหน้าที่ เช่น
schoolhub-settings-single-modal-final-script.js) ถูกแยกเป็นไฟล์เล็ก ๆ ตามเดิม

วิธีใช้งาน
----------
เปิดไฟล์ index.html ผ่านเว็บเซิร์ฟเวอร์ (ไม่ใช่เปิดตรง ๆ แบบ file://)
เนื่องจากมี <script type="module"> ซึ่ง browser จะบล็อกการโหลดไฟล์ local
ผ่าน file:// ตัวอย่างเช่น:

  python3 -m http.server 8000

แล้วเปิด http://localhost:8000/index.html
